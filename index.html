Introdução

Objetivo: Aprender a criar sites utilizando prompts de IA para gerar código HTML, CSS e JavaScript de maneira prática e eficiente.

  • Por que usar IA? A IA, como o Grok da xAI, pode acelerar o desenvolvimento, reduzindo o tempo de codificação e permitindo que iniciantes criem sites profissionais.
  • O que você vai criar: Um site educacional com cabeçalho fixo, menu lateral interativo, conteúdo detalhado e rodapé fixo.
  • Ferramentas necessárias: Uma IA (ex.: Grok), um editor de texto (ex.: VS Code) e um navegador web (ex.: Chrome).

Benefícios: Mesmo sem experiência avançada, você pode criar sites funcionais e visualmente atraentes usando prompts bem elaborados.

O que é um Prompt?

Um prompt é uma instrução clara e específica fornecida à IA para obter uma resposta útil, como um trecho de código ou um design de site.

Características de um Bom Prompt:

  • Clareza: Use linguagem direta, evitando termos vagos como "bonito". Exemplo: "Use fundo branco e texto vermelho."
  • Contexto: Explique o objetivo, como o tipo de site (portfólio, blog, etc.). Exemplo: "Crie um site educacional."
  • Especificidade: Detalhe layout, cores, e funcionalidades. Exemplo: "Inclua um menu vertical com botões de 160px por 40px."

Exemplo de Prompt: "Crie um site em HTML/CSS com um menu vertical à esquerda, cores vermelhas (#B22222), e um cabeçalho fixo com logo."

Dica: Pense no prompt como uma receita: cada ingrediente (detalhe) deve ser claro para garantir o resultado esperado.

Passo 1: Criando o Cabeçalho

O cabeçalho é fixo no topo, com o logo da FIEC à esquerda e o título à direita. Ele usa a cor vermelha (#B22222) para consistência visual.

Prompt: "Crie um cabeçalho em HTML/CSS com logo à esquerda, título à direita, fundo branco, borda vermelha (#B22222). O logo está em https://www.fiec.com.br/site/imgs/logo-fiec-vermelho.png. Fixo no topo."

<header style="background: white; padding: 1rem; display: flex; align-items: center; justify-content: space-between; border-bottom: 2px solid #B22222; position: fixed; top: 0; width: 100%; z-index: 1000;">
    <div class="logo">
        <img src="https://www.fiec.com.br/site/imgs/logo-fiec-vermelho.png" alt="FIEC Logo" style="height: 60px;">
    </div>
    <div class="title" style="color: #B22222; font-size: 1.5rem; font-weight: bold;">
        Desenvolvendo Prompts para IA no Desenvolvimento Web
    </div>
</header>

Explicação:

  • position: fixed: Mantém o cabeçalho no topo ao rolar a página.
  • display: flex: Alinha o logo e o título horizontalmente.
  • z-index: 1000: Garante que o cabeçalho fique acima de outros elementos.

Dica: Teste o código em um navegador para verificar se o cabeçalho permanece fixo e visível.

Visual: Captura de tela do cabeçalho, destacando o logo à esquerda, título à direita, fundo branco e borda vermelha (#B22222).

Passo 2: Criando o Menu Lateral

O menu é vertical, fixo (sticky) para acompanhar o scroll, com botões de mesmo tamanho e interações visuais (hover e clique).

Prompt: "Crie um menu vertical em HTML/CSS/JS, à esquerda, com 5 itens (Introdução, O que é um Prompt?, Prompts para Web, Boas Práticas, Exemplos Práticos). Botões de 160px por 40px, bordas vermelhas (#B22222), hover vermelho, clique vermelho escuro (#8B0000). Sticky, 100px abaixo do topo."

<nav class="menu" style="background: white; padding: 1rem; display: flex; flex-direction: column; width: 200px; border-right: 2px solid #B22222; position: sticky; top: 100px; height: calc(100vh - 100px);">
    <a href="#introducao" style="width: 160px; min-height: 40px; border: 1px solid #B22222; border-radius: 4px; color: #B22222; text-decoration: none; padding: 0.5rem 1rem; font-size: 0.9rem; display: flex; align-items: center; justify-content: center; margin-bottom: 0.5rem;">Introdução</a>
    <!-- Outros itens -->
</nav>
<style>
.menu a:hover { background-color: #B22222; color: white; border-color: #B22222; }
.menu a:active, .menu a.clicked { background-color: #8B0000; color: white; border-color: #8B0000; }
</style>
<script>
document.querySelectorAll('.menu a').forEach(item => {
    item.addEventListener('click', () => {
        document.querySelectorAll('.menu a').forEach(link => link.classList.remove('clicked'));
        item.classList.add('clicked');
    });
});
</script>

Explicação:

  • position: sticky: Faz o menu acompanhar o scroll, começando 100px abaixo do topo.
  • width: 160px; min-height: 40px: Garante botões uniformes.
  • transition: Suaviza mudanças de cor no hover e clique.
  • JavaScript: Adiciona a classe clicked ao item selecionado.

Dica: Clique nos botões no navegador para verificar as interações de hover e clique.

Visual: Diagrama mostrando o menu com cinco botões verticais, destacando o estado normal (borda vermelha #B22222), hover (fundo vermelho #B22222) e clique (fundo vermelho escuro #8B0000).

Passo 3: Criando o Conteúdo Principal

O conteúdo principal contém cinco seções com textos, listas e exemplos de código, posicionados à direita do menu, com largura máxima de 800px.

Explicação:

Boas Práticas para Prompts

Para criar prompts eficazes, siga estas práticas para obter os melhores resultados da IA:

  • Especifique Tecnologias: Indique claramente HTML, CSS, JavaScript, etc. Exemplo: "Use apenas HTML e CSS."
  • Detalhe o Layout: Descreva a posição dos elementos. Exemplo: "Menu vertical à esquerda, conteúdo à direita."
  • Inclua Referências: Forneça links para logos ou sites inspiradores. Exemplo: "Baseie-se no logo em https://www.fiec.com.br/site/imgs/logo-fiec-vermelho.png."
  • Itere: Ajuste o resultado com prompts específicos. Exemplo: "Adicione bordas vermelhas aos botões do menu."
  • Teste Sempre: Valide o código no navegador para garantir funcionalidade.

Por que isso importa? Um prompt bem elaborado economiza tempo e reduz erros, produzindo resultados mais próximos do esperado.

Erros Comuns a Evitar

Evite os seguintes erros ao usar prompts para garantir resultados úteis:

  • Prompts Vagos: Ex.: "Crie um site bonito."
    Solução: "Crie um site em HTML/CSS com menu vertical e cores #B22222."
  • Falta de Iteração: Não revisar o código gerado.
    Solução: Peça ajustes, como "Corrija o alinhamento do menu."
  • Ignorar Testes: Não validar o código no navegador.
    Solução: Sempre teste cada parte do código para verificar erros.
  • Excesso de Ambiguidade: Não especificar detalhes visuais ou funcionais.
    Solução: Inclua cores, tamanhos e interações específicas.

Técnicas Avançadas de Prompts

Para melhorar ainda mais seus prompts, experimente estas técnicas avançadas:

  • Divida Tarefas: Peça o código em etapas (ex.: HTML primeiro, depois CSS). Exemplo: "Crie a estrutura HTML, depois adicione CSS."
  • Use Exemplos: Inclua trechos de código ou referências visuais. Exemplo: "Baseie o menu em <nav> com botões de 160px."
  • Especifique Responsividade: Exemplo: "Garanta que o menu vire horizontal em telas menores que 768px."
  • Peça Validação: Exemplo: "Verifique se o código segue HTML5 e CSS3 válidos."
  • Combine Estilo e Funcionalidade: Exemplo: "Adicione um menu com hover vermelho e clique interativo via JavaScript."

Atividade Prática

Tarefa: Crie o site da apostila usando os prompts fornecidos, seguindo estas etapas:

  1. Crie um arquivo index.html em um editor de texto (ex.: VS Code).
  2. Use os prompts para gerar o cabeçalho, menu, conteúdo e rodapé, um de cada vez.
  3. Teste cada parte no navegador para verificar o layout e interações.
  4. Ajuste com prompts específicos, como "Adicione espaçamento de 100px entre o menu e o cabeçalho."
  5. Desafio: Adicione uma nova seção com o prompt: "Crie uma seção sobre ferramentas de IA recomendadas, com título vermelho e uma lista de ferramentas."

Exemplo de Código do Desafio:

<section id="ferramentas-ia">
    <h2 style="color: #B22222;">Ferramentas de IA Recomendadas</h2>
    <ul>
        <li>Grok: Criação de código e respostas técnicas.</li>
        <li>ChatGPT: Geração de texto e ideias.</li>
    </ul>
</section>

Dica: Use a apostila como referência e experimente criar seus próprios prompts para personalizar o site.

Dicas para Solução de Problemas

Se algo não funcionar, siga estas dicas para identificar e corrigir problemas:

  • Código Não Funciona: Verifique a sintaxe, como tags HTML não fechadas ou erros em CSS/JavaScript. Exemplo: Certifique-se de que </div> está presente para cada <div>.
  • Layout Errado: Ajuste com prompts específicos, como "Corrija o alinhamento do menu para ficar à esquerda."
  • IA Não Entende: Reformule o prompt com mais detalhes. Exemplo: Em vez de "Melhore o menu", use "Adicione bordas vermelhas de 1px aos botões do menu."
  • Teste Sempre: Use a ferramenta de inspeção do navegador (F12) para identificar erros de CSS ou JavaScript.
  • Exemplo de Correção: Se o menu não for interativo, adicione o JavaScript correto com "Adicione um script para mudar a cor do botão ao clicar."
